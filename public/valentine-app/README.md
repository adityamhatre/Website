## How It Works

- **Desktop**: The "No" button moves away when your mouse gets close to it
- **Mobile**: The "No" button moves to a random position when you tap it
- **Yes Button**: Shows a celebratory popup with an image

## Customization

To customize the popup image, replace `image.png` in the same directory with your own image.
